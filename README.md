### Features

- Create a task
- Complete a task
- Delete a task
- Default Categories and Priorities
- Filter by category
- Message toast notifications
- Drag and drop sorting

## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Delvoid/todo-app.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start dev server
   ```js
   npm run dev
   ```
